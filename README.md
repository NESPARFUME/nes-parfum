# 🌸 NES PARFUM - Parfüm E-Ticaret Sitesi

## Özellikler

✨ **Admin Paneli**: Parfümler ekleyebilir, fotoğraf ve fiyat belirleyebilirsiniz

🛒 **Müşteri Arayüzü**: Müşteriler parfümleri görüp sepete ekleyebilir

📋 **Sipariş Sistemi**: 
- Müşteri bilgilerini girme
- Sipariş özeti görme
- Sipariş onaylama

💬 **WhatsApp Entegrasyonu**: Siparişler direkt WhatsApp'e gönderilir

💾 **LocalStorage**: Tüm veriler tarayıcıda güvenli şekilde saklanır

## Nasıl Kullanılır?

### Admin Olarak:
1. "📋 Admin Paneli" butonuna tıklayın
2. Parfüm adını, fiyatını ve fotoğraf URL'ini girin
3. "✅ Parfümü Ekle" butonuna tıklayın

### Müşteri Olarak:
1. Parfümleri göreceksiniz
2. Miktarı seçip "🛒 Sepete Ekle" yapın
3. Aşağıda müşteri bilgilerinizi girin
4. "⚙️ Siparişi Hazırla" butonuna tıklayın
5. Sipariş özeti göreceksiniz
6. "✅ Onayla ve Siparişi Gönder" tıklayın
7. Sipariş otomatik WhatsApp'e gönderilecek
